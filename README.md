# 🏛️ ModernStudio – Premium Architecture Studio Website

## 📌 Project Overview
**ModernStudio** is a high-end, minimalist, and fully responsive architecture studio website developed using semantic **HTML5 and CSS3**. The website showcases state-of-the-art futuristic structural designs and eco-friendly buildings in an attractive and user-friendly interface.

The project demonstrates front-end web development core concepts including responsive design, interactive UI components, structured semantic layouts, custom CSS Grid/Flexbox component distribution, smooth scrolling, and premium color theory matching professional architecture portfolio design standards.

---

## 🚀 Features
* 🏠 **Responsive landing page:** Clean and aesthetic landing page tailored for modern design firms.
* 🎨 **Modern and attractive UI design:** Built with a premium color palette (Mocha Mousse, Moonlit Grey, and Ethereal Blue).
* 🌿 **Sustainable concepts section:** Interactive design showcase for futuristic smart buildings.
* 🛠️ **Advanced component layouts:** Custom-designed interactive Hero split-view and grid-based Services display.
* 📱 **Mobile-friendly navigation menu:** Flawless visual adaptation and responsive navigation across all screen sizes.
* 📬 **About and Contact sections:** A professional studio introduction coupled with a structured message form.
* 🗂️ **Modern 3-column footer:** Premium layout distributing studio info, quick navigational links, and social media networks.
* 💬 **Client inquiry terminal:** Interactive input fields for full name, email, and project messages.
